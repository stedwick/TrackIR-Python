# TrackIR C++ and Python SDK

This is a C++/Python SDK for integrating headtracking data from TrackIR into your projects. The SDK includes three sample projects, one demonstrating how to stream the headtracking data to a console application, and the other showing how to create a basic first-person perspective camera app using OpenGL, and a last one with a simple GUI in python.

# Getting Started
First install Visual Studio 2022, with Python and MFC and ATL for v143 (x86 and x64). You can install these components in the "individual components" tab of the Visual Studio Installer.

To use this SDK, you will need a TrackIR headset and the TrackIR software installed on your computer. You can download the TrackIR software from the website: https://www.trackir.com/.

To get started, download the SDK to your local machine from https://www.trackir.com/developers. Then, open one of the sample projects in Visual Studio and build/run the project. You should see the headtracking data streaming to the console or the first-person camera app.

# Using the SDK

To use the SDK in your own projects, simply include the necessary headers and link against the provided libraries. The SDK provides functions for accessing the headtracking data, including the position and orientation of the user's head.

Be sure to have TrackIR running and updated before loading any samples.

Refer to the sample projects source code and comments for how to implement the SDK in your own projects. 

It is highly reccomended you read our documentation at https://docs.trackir.com/
